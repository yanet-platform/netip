# Обозначения

В этом разделе собраны все обозначения, принятые в книге.

## Адреса и пространство

| Обозначение | Смысл |
|---|---|
| $n$ | длина адреса в битах: $n = 32$ для IPv4, $n = 128$ для IPv6 |
| $\{0,1\}^n$ | множество всех $n$-битных адресов (булево пространство) |
| $X[i]$ | $i$-й бит адреса $X$, нумерация слева направо от $0$ до $n-1$; $X[0]$ — старший бит |
| $\text{int}(X)$ | целочисленная интерпретация битовой строки $X$ в big-endian порядке |

## Сети

| Обозначение | Смысл |
|---|---|
| $(a, m)$ | нормализованная сеть: $a$ — адрес, $m$ — маска; оба в $\{0,1\}^n$ |
| $S(a, m)$ | множество адресов сети: $S(a, m) = \{ X \in \{0,1\}^n : X \land m = a \}$ |
| $a \land m = a$ | инвариант нормализации: свободные биты адреса обнулены |
| $k$ | длина префикса контигуальной маски; сеть с маской $1^k 0^{n-k}$ записывается как `a/k` |
| $\overline{m}$, $\lnot m$ | побитовое отрицание маски $m$; в теореме 1c обозначается $\overline{m}$ (hostmask), в главах 2–3 — $\lnot m$ |

## Побитовые операции

| Обозначение | Rust / C | Смысл |
|---|---|---|
| $X \land Y$ | `x & y` | побитовое И (AND) |
| $X \lor Y$ | `x \| y` | побитовое ИЛИ (OR) |
| $\lnot X$ | `!x` | побитовое НЕ (NOT) |
| $X \oplus Y$ | `x ^ y` | побитовое исключающее ИЛИ (XOR) |

Все операции действуют покоординатно: $(X \land Y)[i] = X[i] \cdot Y[i]$, и т.д.

## Характеристики масок и сетей

| Обозначение | Смысл |
|---|---|
| $\text{popcount}(x)$ | число единичных бит в $x$ |
| $\text{trailing\_zeros}(x)$ | число нулевых бит с конца (младших) |
| $\text{trailing\_ones}(x)$ | число единичных бит с конца |
| $2^{n - \text{popcount}(m)}$ | мощность сети $(a, m)$ — число адресов |
| $h$ | индекс хоста внутри сети, $0 \leq h < |S(a,m)|$ |
| $\text{addr}(h)$ | $h$-й адрес сети: $\text{addr}(h) = a \lor \text{pdep}(h, \lnot m)$ |
| $\text{pdep}(h, x)$ | parallel bit deposit: распределяет биты $h$ по единичным позициям $x$ |

## Множества и порядок

| Обозначение | Смысл |
|---|---|
| $A \subseteq B$ | включение: $S(a_1, m_1) \subseteq S(a_2, m_2)$ |
| $A \cap B$ | пересечение множеств адресов |
| $A \cup B$ | объединение множеств адресов |
| $A \setminus B$ | разность множеств адресов |
| $A \sqcup B$ | дизъюнктное объединение (слагаемые попарно не пересекаются) |
| $\varnothing$ | пустое множество |
| $\lfloor x \rfloor$ | целая часть от $x$ (floor) |

## Операции над сетями

| Операция | Результат | Раздел |
|---|---|---|
| $A \cap B$ | сеть $(a_1 \lor a_2,\ m_1 \lor m_2)$ или $\varnothing$ | 3.1 |
| $A \setminus B$ | набор до $\text{popcount}(m_2 \land \lnot m_1)$ сетей | 3.3 |
| $\text{supernet}(A, B)$ | наименьшая сеть, содержащая $A \cup B$ | 3.4 |
| $\text{merge}(A, B)$ | единственная сеть, равная $A \cup B$, или $\varnothing$ | 3.5 |

## Алгебра

| Обозначение | Смысл |
|---|---|
| $GF(2)$ | поле из двух элементов; сложение — XOR, умножение — AND |
| $GF(2)^n$ | векторное пространство над $GF(2)$ (отождествляется с $\{0,1\}^n$) |
| $V_m$ | линейное подпространство $\{ v \in GF(2)^n : v \land m = 0 \}$ |
| $a + V_m$ | аффинное подпространство (сдвиг $V_m$ на вектор $a$); равно $S(a, m)$ |
| $e_i$ | стандартный единичный вектор: единица в позиции $i$, нули остальных |

## Геометрия

| Обозначение | Смысл |
|---|---|
| $\text{Face}(P, v)$ | грань гиперкуба: фиксированы координаты $P$ со значениями $v$ |
| $d_H(X, Y)$ | расстояние Хэмминга: число позиций, в которых $X$ и $Y$ различаются |

## Доказательства

| Символ | Смысл |
|---|---|
| $\blacksquare$ | конец доказательства |
| $\checkmark$ | подусловие проверено |
| $\Rightarrow$, $\Leftarrow$, $\iff$ | импликация и эквивалентность |
